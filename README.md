# weather-app
A responsive weather forecasting app built with modern web technologies, providing real‑time temperature, humidity, and forecast data using OpenWeather API. Designed with a clean UI and scalable code structure for seamless user experience.

  
It provides real-time weather updates including temperature, humidity, wind speed, and 5-day forecast using the OpenWeather API.  

## ✨ Features
- 🌦️ Real-time weather data for any city
- 📱 Responsive design for mobile and desktop
- 🔍 Search functionality with instant results
- 🎨 Clean and user-friendly UI
- ⚡ Fast performance with optimized code structure

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- React.js (for UI components)
- OpenWeather API (for live weather data)
- GitHub Pages (for hosting)

## 🚀 Getting Started
1. Clone the repository  
   ```bash
   git clone https://github.com/harsh2874/weather-app.git

2. Navigate to the project folder
   cd weather-app

3. Install dependencies
   npm install

4. Run the app locally
   npm start
